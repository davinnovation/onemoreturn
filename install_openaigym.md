# Install [OpenAI gym](https://gym.openai.com)

@reinforcement_learning @install

## on Ubuntu
    
    | Prerequisite
    - sudo apt-get install swig
    
    | Install gym & environments
    1. ```git clone https://github.com/openai/gym```
    2. ```cd gym```
    3. ```pip install -e .``` or ```pip install -e .[all]```
